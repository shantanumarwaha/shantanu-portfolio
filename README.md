# Shantanu Marwaha — Portfolio

Personal recruiting portfolio built with Next.js and Tailwind CSS.

## Editing content

All copy (name, executive summary, career roadmap, goals, and projects) lives in
[`lib/content.ts`](./lib/content.ts). Update that file to change any text on the site.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Build

```bash
npm run build
npm run start
```

## Deploy

The site is a standard Next.js app and deploys as-is to [Vercel](https://vercel.com/new).
